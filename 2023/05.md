## 2023-05-02
1. vue3 에서 `ref` 사용시 .value 를 사용해서 값을 변경해야함.

```js
<script setup>
import { ref } from 'vue';

let obj = ref({ count: 0 });
const onClick = () => {
  obj.value.count++;
};
const onChange = () => {
  obj.value = {
    count: 100
  };
}
</script>

<template>
  <div>
    {{ obj.count }}
    <button @click="onClick">증가</button>
    <button @click="onChange">변경</button>
  </div>
</template>
```

## 2023-04-04
1. `nuxt` 를 이용해서 노드 모듈을 설치한 뒤, `npm install --save net tls` 다음과 같은 모듈을 설치하라는 오류가 발생했을 때 해결 방법으로 모듈을 설치해주지 말고, 의존성 충돌 문제일 가능성이 높으므로 아래와 같이 `nuxt.config.js` 를 설정해줍니다.

```js
  build: {
    extend(config, { isDev, isClient }) {
      config.node = {
        fs: 'empty',
        net: 'empty',
        tls: 'empty',
      }
    },

```

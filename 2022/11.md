2. vuex 에서 actions, mutations 사용
- 모든 api 는 actions 에서 사용함. 그런 뒤, mutations 를 이용하여 state 를 커밋해줌.
- actions 는 dispatch, mutations 는 commit 임.
- getters 를 만들어 state 를 리턴해주는데, mapGetter 를 사용하면 좀 더 깔끔하게 가능함. (참고)
